ES的目录结构

  • bin
    脚本文件，包括启动elasticsearch,安装插件。运行统计数据等

  • config
    配置文件: elasticsearch.yml
    集群配置文件,user,role based相关配置

  • JDK
    java运行环境

  • data
    配置文件:path.data
    数据文件

  • lib
    java类库

  • logs
    配置文件:path.log
    日志文件

  • modules
    包含所有es模块

  • plugins
    包含所有已安装插件
