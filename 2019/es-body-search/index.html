Query String Query

Simple Query String Query

  • 类似Query String,但是会忽略错误的语法，同时只支持部分查询语法

  • 不支持AND OR NOT,会当作字符串处理

  • Term之间默认的关系是OR,可以指定Operator

  • 支持部分逻辑 

    + 替代 AND  
| 替代 OR  
- 替代 NOT
